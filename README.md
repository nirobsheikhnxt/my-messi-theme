# Leo Messi Edition Discord Theme

A premium, customized Vencord theme styled around Lionel Messi's classic color palette (burgundy crimson, navy, and gold) using the Dark Matter theme structure.

## 🚀 Quick Setup Instructions

1. Upload your wallpaper file `Wallpaper (1).jpg` to this GitHub repository.
2. Open the image file on GitHub, copy its raw URL.
3. Edit `custom-messi.theme.css` and paste the raw image link into the `--background-image` variable.
4. Copy the raw link of the updated `custom-messi.theme.css` file.
5. Paste the CSS link in the **Online Themes** input field inside your **Vencord Settings > Themes** tab and click **Add**.
